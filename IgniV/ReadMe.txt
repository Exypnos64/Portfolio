This is a nearly-fully functional programming language written in Java the fall semester of 2021 in a programming languages course.

To run it, the following command is recommended to be run:
cd "C:\*folder location path*\IgniV\src\IgniV\" && javac * -d "C:\*folder location path*\IgniV\build\classes\igniv"  && cd "C:\*folder location path*\IgniV\build\classes\igniv" && java IgniV/IgniV "C:\*folder location path*\IgniV\*filename*.igv"
